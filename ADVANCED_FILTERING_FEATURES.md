# Advanced Profile Download Filtering Features

## Overview
The profile download feature now includes comprehensive filtering options to help you find and download exactly the content you want from any supported platform.

## Supported Platforms
- **TikTok** - Full engagement data support
- **YouTube** - Full engagement data support  
- **Facebook** - Full engagement data support
- **Instagram** - Basic filtering support
- **Twitter/X** - Basic filtering support
- **Other platforms** - Basic view count filtering

## Filter Categories

### 1. Engagement Filters
Filter videos based on social engagement metrics:

- **View Count Range**: Min/Max views (e.g., "1M" to "10M")
- **Like Count Range**: Min/Max likes (e.g., "50K" to "500K") 
- **Comment Count Range**: Min/Max comments (e.g., "100" to "5K")
- **Share Count Range**: Min/Max shares/reposts (e.g., "50" to "1K")

### 2. Content Filters
Filter videos based on content characteristics:

- **Duration Range**: Min/Max duration in seconds (e.g., "30" to "300")
- **Verified Creators Only**: Only download from verified/checkmarked accounts
- **Trending Content Only**: Only download videos marked as trending
- **Exclude Short Videos**: Skip videos under 60 seconds
- **Exclude Long Videos**: Skip videos over 10 minutes

### 3. Sorting Options
Enhanced sorting options for better content discovery:

- **Newest First**: Most recent uploads first
- **Oldest First**: Earliest uploads first  
- **Most Popular**: Highest view count first
- **Most Liked**: Highest like count first
- **Most Commented**: Highest comment count first
- **Most Shared**: Highest share count first

## How to Use

### Via Settings Dialog
1. Click the **Settings** button in Profile Download tab
2. Navigate through the three tabs:
   - **Basic Settings**: Video count, sorting, quality
   - **Advanced Filters**: Engagement and content filters
   - **Download Options**: Audio, subtitles, file organization
3. Configure your desired filters
4. Click **Start Download**

### Filter Input Formats
- **Numbers**: Use raw numbers (e.g., "1000000") 
- **K Suffix**: Use "K" for thousands (e.g., "50K" = 50,000)
- **M Suffix**: Use "M" for millions (e.g., "1.5M" = 1,500,000)
- **Duration**: Always in seconds (e.g., "300" = 5 minutes)

## Filter Persistence
All filter settings are automatically saved and restored between sessions. Your preferred filters will be remembered for future downloads.

## Performance Tips
- Use engagement filters to reduce processing time
- Combine multiple filters for precise targeting
- Start with broader filters and refine as needed
- Monitor filter results in the download log

## Troubleshooting

### No Videos Match Filters
- Check if filter criteria are too restrictive
- Verify the profile has content matching your criteria
- Try broader ranges or fewer simultaneous filters

### Missing Engagement Data
- Some platforms provide limited data
- Older videos may have incomplete metrics
- Private or restricted content may lack data

### Filter Not Working
- Ensure correct input format (use K/M suffixes)
- Check that the platform supports the specific filter
- Verify profile extraction was successful
